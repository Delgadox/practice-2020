Для установки данного проекта:
1. Склонировать его из данного репозитория.
2. Прописать в консоле composer u и npm i
3. Подготовить Базу Данных, для этого вам нужно изменить настройки фаила .env. После чего прописать в консоле php artisan migrate
4. После чего прописать в консоле php artisan serve
